¿Qué es? 
La capa que RECIBE las peticiones HTTP y las transforma en llamadas a servicios.

En js seria:

// controllers/tratamientoController.js
class TratamientoController {
  
  // Maneja POST /api/tratamientos
  async crear(req, res) {
    try {
      // 1. Extraer datos del body de la petición
      const datos = req.body;
      
      // 2. Llamar al servicio (lógica de negocio)
      const nuevoTratamiento = await tratamientoService.crearTratamiento(datos);
      
      // 3. Enviar respuesta exitosa
      res.status(201).json({
        success: true,
        message: 'Tratamiento creado',
        data: nuevoTratamiento
      });
      
    } catch (error) {
      // 4. Manejar errores
      res.status(400).json({
        success: false,
        message: error.message
      });
    }
  }
  
  // Maneja GET /api/tratamientos
  async listar(req, res) {
    const tratamientos = await tratamientoService.listarTodos();
    res.json(tratamientos);
  }
}

¿Qué va aquí?

- Extraer parámetros de la URL (req.params)
- Extraer datos del body (req.body)
- Extraer query strings (req.query)
- Llamar a servicios
- Enviar respuestas HTTP (JSON o HTML)
- Manejar errores HTTP (404, 400, 500)

¿Qué NO va aquí?

- Consultas SQL
- Reglas de negocio complejas
