¿Qué es? Clases que representan UNA fila de tu base de datos. 
Definen la estructura de tus datos.

Por ejemplo en js seria:

// models/Tratamiento.js

class Tratamiento {
  constructor(data = {}) {
    this.idtratamiento = data.idtratamiento;
    this.nombretratamiento = data.nombretratamiento;
    this.precio = data.precio;
    this.duracion = data.duracion;
  }

  // Método de validación (reglas del negocio PURAS)
  validate() {
    if (!this.nombretratamiento) throw new Error('Nombre requerido');
    if (this.precio <= 0) throw new Error('Precio debe ser positivo');
  }
}

¿Qué va aquí?

- Clases que mapean a tablas (Tratamiento, Cliente, Cita)
- Validaciones básicas del objeto (no reglas de negocio complejas)
- Métodos de formateo (toJSON, calcularEdad, etc.)
